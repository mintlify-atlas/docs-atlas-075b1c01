The Tub Harness is a macOS SwiftUI app that bridges live audio input and an external ML model server. It runs a 10 Hz frame loop, packages audio features into protocol v2 packets, applies mode-specific DSP to the live audio graph, and logs a training substrate to JSONL for each session.

## How it works

<Steps>
  <Step title="Start the model server">
    Run `tub-ml serve` from the companion ML repo. The harness connects over UDP on port `9910` by default.
  </Step>
  <Step title="Open the Xcode project">
    Open `TheTubHarness/TheTubHarness.xcodeproj` and run the `TheTubHarness` scheme targeting macOS.
  </Step>
  <Step title="Select your input">
    In the 4-panel UI, use the **Input** picker to choose your microphone or line input device. Select the **Audio + Real Features** run profile.
  </Step>
  <Step title="Choose a mode and run">
    Pick a DSP mode (0–10), press **Start**, and use the **JOLT** and **CLEAR** buttons to interact with the ML model in real time.
  </Step>
</Steps>
